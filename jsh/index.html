The Open-Source JavaScript Shell

About:

JSH is a xTerm-based JavaScript shell made for hobbists and students to learn JavaScript.

With it's lightweight nature and reliance on the browser, it is powerful and flexable. JSH is most

importantly embeddable. You could embed it into any application as you can see here.

Download

You can download the latest release version of JSH by clicking the very obvious "download latest release" button, or if you're feeling a bit SPICY, you can download a a pre-release version of JSH to test it out.
